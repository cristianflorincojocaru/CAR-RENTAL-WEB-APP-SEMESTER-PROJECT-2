# DISTRIBUTED-INFORMATION-SYSTEM-FOR-INVENTORY-AND-SALES-MANAGEMENT-FOR-A-CAR-RENTAL-COMPANY
Full-stack web application built with Angular (front-end), ASP.NET (back-end), and SQL Server Express (database). The system provides a responsive user interface, secure RESTful APIs, structured data management, and a scalable architecture for inventory and sales management in a car rental company.
